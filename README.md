# 💰 Interest Calculator

A simple and user-friendly web application to calculate **Simple Interest** based on user inputs.

## 🚀 Features

* Calculate interest using:

  * Principal Amount
  * Rate of Interest
  * From Date and To Date
* Displays:

  * Total Interest
  * Total Amount
  * Number of days
* Clean and mobile-friendly UI
